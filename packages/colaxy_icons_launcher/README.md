# colaxy_icons_launcher

This project is a fork of [icons_launcher](https://github.com/mrrhak/icons_launcher) by [mrrhak](https://github.com/mrrhak).

## Changes
The only change made in this version is the addition of **SVG image support**. All other features and functionalities remain identical to the original library.
